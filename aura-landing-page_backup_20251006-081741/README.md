# AURA Landing Page

## Installation
```bash
npm install
```

## Development
```bash
npm start
```

## Build for Production
```bash
npm run build
```

## Deploy
- Upload `build/` folder to any static hosting
- Compatible with Vercel, Netlify, AWS S3, etc.

## Features
- 100% React frontend
- Bootstrap 5 responsive
- TikTok Glass theme
- No backend dependencies
- SEO optimized
