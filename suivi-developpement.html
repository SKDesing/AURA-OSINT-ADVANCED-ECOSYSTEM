Architecture Unifiée

  • Frontend (Port 3000): Dashboard React, Chat IA, Outils OSINT
  • Backend (Port 4011): Express API, WebSocket, IA Qwen, 17 outils OSINT
  • Bases de Données: PostgreSQL, Redis, Elasticsearch, Qdrant

Guide de Déploiement

Instructions de lancement

⚡ Lancement Rapide

# 1. Démarrage unifié (Port 4011)
./RUN-AURA-UNIFIED.sh

# 2. Site vitrine
cd marketing/sites/vitrine-aura-advanced-osint-ecosystem
npm install && npm start

# 3. Tests complets
./test-frontend-complete.sh

🔧 Configuration Requise

  • Docker: Containerisation complète
  • Ports: 4011 (API & IA), 3000 (Site vitrine), 5432 (PostgreSQL)
  • Dépendances: Node.js 18+, Python 3.9+, React 18
